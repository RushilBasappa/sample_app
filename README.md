# Twitter Clone

Project Period: 12th Feb,2016 - 7th Mar,2016

## Steps to run
```
git clone git@github.com:RushilBasappa/twitter-clone.git
--> install ruby v2.2.2
gem install bundler
bundle install
bundle exec rake db:migrate
bundle exec rails server
```
